#  Login Page - Dark Blur Glassmorphism UI

A high-performance, responsive single-page authentication portal built using Vanilla HTML5, CSS3, and JavaScript. Features modern **Glassmorphism**, ambient background light animations, interactive floating form labels, and zero external framework dependencies.

---

##  Key Features

* **Glassmorphic Aesthetic:** Frosted glass background using CSS `backdrop-filter`.
* **3-in-1 Dynamic Switcher:** Seamless state transitions between Login, Sign Up, and Forgot Password views.
* **Micro-Animations:** Pulsing backdrop ambient orbs and smooth input state transitions.
* **Single-File Architecture:** Built with pure native Web Standards (HTML5, embedded CSS3, Vanilla JS) without external dependencies.
* **Responsive Design:** Optimized for mobile, tablet, and desktop viewports.
